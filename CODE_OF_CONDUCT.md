# Code of Conduct

ScreenTutor aims to be a welcoming, safe, and constructive open-source project.

## Expected behavior

- Be respectful of different backgrounds, experience levels, and viewpoints.
- Give specific, actionable feedback about the work rather than the person.
- Assume good intent, ask clarifying questions, and acknowledge mistakes.
- Respect privacy, especially when examples may contain screen or voice data.
- Keep discussions relevant to improving the project and its community.

## Unacceptable behavior

Harassment, threats, discrimination, sexualized language or imagery, personal attacks, deliberate intimidation, sustained disruption, and publishing another person's private information are not tolerated.

## Scope and enforcement

This code applies in repository discussions, issues, pull requests, reviews, and other spaces where someone represents the project. Maintainers may edit or remove content, reject contributions, or temporarily or permanently restrict participation when conduct conflicts with these standards.

For moderation concerns, contact the maintainer privately using the contact information on the [maintainer's GitHub profile](https://github.com/SebastianBoehler). Do not post sensitive incident details publicly. Reports will be reviewed as promptly and confidentially as practical.
